RTP/RTSP/RTC

RTP = Real-Time Transport Protocol
- Audio/Video-Übertragung über Pakete (üblicherweise UDP)
- RFC 1889

RTSP = Real-Time Streaming Protocol
- Steuerung der Übertragung von audiovisuellen Datenströmen
- HTTP-basiert
- RFC 2326

RTC = Real-Time Control Protocol
- Sicherstellung von QoS bei der Übertragung
- z.B. Synchronisation, Steuerung der Bandbreite

SRTP = Secure RTP
